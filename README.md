# kis-linux

### Задача 1. Phonebook module
Исходники модуля встроены в дерево исходников Linux. Задача оформлена в виде [патча](0001-Task-1-add-phonebook-module.patch). Общение с модулем происходит через `/dev/phonebook`.
Пример тестового скрипта взаимодействия можно найти в корне под названием `test_phonebook.sh`.

### Задача 2. Keyboard module
Исходники модуля в [соответствующей папке](keyboard_module/). Собранный модуль лежит в корне образа ramfs.
Модуль выводит результат в логи раз в минуту.

### Задача 3. Новое поле в task_struct
Задача оформлена в виде [патча](0001-Task-3-add-new-field-in-task-struct.patch). Поле называется `cnt_planned`.
Скрипт с примером расположен в корне и называется `test_task_struct.sh`.

### Задача 4. mmaneg module
Исходники модуля в [соответствующей папке](mmaneg_module/). Собранный модуль лежит в корне образа. Общение происходит через `/proc/mmaneg`.
Пример можно найти в тестовом скрипте `test_mmaneg.sh` в корне. Для удобства добавлена ещё команда `readval`.
